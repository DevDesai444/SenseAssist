# Code of Conduct

This project follows a professional and respectful collaboration model.

## Expected behavior

- Be respectful and constructive.
- Focus feedback on code and design, not people.
- Assume good intent and ask clarifying questions before escalating.
- Keep discussions technical, specific, and actionable.

## Unacceptable behavior

- Harassment, abuse, or discriminatory language.
- Personal attacks, insults, or threats.
- Publishing private information without consent.
- Deliberate disruption of project discussions or reviews.

## Enforcement

Maintainers may edit, remove, or moderate content that violates this code and may restrict participation for repeated violations.

## Reporting

Report conduct concerns through repository maintainers using a private channel where available. If no private channel is available, open an issue requesting one without posting sensitive details.
